== NAME
system_prefetch - system control

== SYNOPSIS
[verse]
____
*#define SYSTEM_FREQUENCY 16777216* // ~16.78 MHz

**void system_prefetch(bool **__enable__**);**
____

== DESCRIPTION
The *system_prefetch*() function enables or disables the GBA's prefetch
buffer, based on the value of _enable_. When enabled, the prefetch
buffer typically improves the performance of code living in ROM. During
startup, the prefetch buffer is enabled by the library.
