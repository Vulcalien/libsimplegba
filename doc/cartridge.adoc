== NAME
cartridge_header, cartridge_missing - game cartridge

== SYNOPSIS
[verse]
____
*#include <libsimplegba.h>*

*const struct CartridgeHeader +++*+++cartridge_header(void);*
*bool cartridge_missing(void);*
____

== DESCRIPTION
The *cartridge_header*() function returns a pointer to the cartridge's
header. The header structure is defined as follows:

[source,c]
----
struct CartridgeHeader {
    u32 entry_point;
    u8  logo[156];

    char game_title[12];
    char game_code [4];
    char maker_code[2];

    u8 fixed_value; // 0x96

    u8 main_unit_code;
    u8 device_type;

    u8 reserved_1[7];

    u8 software_version;
    u8 checksum;

    u8 reserved_2[2];
};
----

The *cartridge_missing*() function returns true if the game cartridge is
missing, false otherwise. To determine the result, the function looks
for a fixed value in the cartridge header.
